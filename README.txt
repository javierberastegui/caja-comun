Instalación:
1. Sube la carpeta economia-pro o el ZIP desde Plugins > Añadir nuevo
2. Activa el plugin
3. Crea una página y añade el shortcode [economia_dashboard]
4. Configura la contraseña y la página desde el panel del plugin
